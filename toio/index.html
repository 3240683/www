操作方法・キーボードでの操作

🎮 コントローラー操作

  • 左スティック: 前進 / 後退 / 旋回
  • A ボタン: LED緑点灯 ＋ 接続音
  • B ボタン: LED赤点灯 ＋ ダメージ音
  • X ボタン: LED青点灯 ＋ レベルアップ音
  • Y ボタン: LED黄点灯 ＋ コイン音

⌨️ キーボード操作 (フォールバック)

  • W / ↑ : 前進
  • S / ↓ : 後退
  • A / ← : 左旋回
  • D / → : 右旋回
  • 1, 2, 3, 4: LEDカラーチェンジ＆効果音
