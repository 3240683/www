# toio & Switch Controller コネクター

このプロジェクトは、Nintendo Switchのコントローラー（Joy-Con または Proコントローラー）を使用して、ブラウザ（Google Chrome / Microsoft Edge）経由で **toio™ コア キューブ** をワイヤレス操縦するための JavaScript アプリケーションです。

Web Bluetooth API と Web Gamepad API を利用しているため、複雑なドライバーのインストールやライブラリのビルド環境を用意することなく、簡単に動作させることができます。

---

## 必要なもの
1. **toio™ コア キューブ**
2. **Nintendo Switch コントローラー** (Joy-Con または Switch Pro コントローラー)
3. **Bluetooth を搭載した Windows PC**
4. **Google Chrome** または **Microsoft Edge** ブラウザ (Web Bluetooth API が有効なブラウザ)
5. **Node.js** (簡易サーバーの起動に使用)

---

## 接続と起動の手順

### ステップ 1: コントローラーを Windows PC にペアリングする
1. Windows の「設定」 > 「Bluetooth とその他のデバイス」を開き、Bluetooth をオンにします。
2. コントローラーをペアリングモードにします。
   - **Joy-Con:** スライドレールの間にある「シンクロボタン」を長押しして、ランプが往復点滅するのを待ちます。
   - **Proコントローラー:** USB端子横の「シンクロボタン」を長押しします。
3. Windows の「デバイスの追加」からコントローラー（`Joy-Con (L)`、`Joy-Con (R)`、または `Pro Controller`）を選択し、ペアリングを完了させます。

### ステップ 2: ローカルWebサーバーを起動する
Web Bluetooth API は、セキュリティ保護された接続（`https` または `localhost`）でのみ動作します。そのため、同梱の簡易Webサーバーを起動します。

1. コマンドプロンプトまたは PowerShell を起動し、このプロジェクトのフォルダに移動します。
   ```bash
   cd "C:\Users\3yuma\OneDrive\デスクトップ\Antigrubity_作ったもの\Switch_Controler"
   ```
2. 以下のコマンドを実行してサーバーを立ち上げます。
   ```bash
   node server.js
   ```
3. 画面に以下が表示されれば成功です。
   ```
   ==================================================
    toio & Switch Controller Server is running!
    URL: http://localhost:3000
   ==================================================
   ```

### ステップ 3: ブラウザで開いて操作する
1. **Google Chrome** または **Microsoft Edge** を開き、 [http://localhost:3000](http://localhost:3000) にアクセスします。
2. PCにペアリングした Switch コントローラーのいずれかのボタン（A, B, X, Yなど）を押します。
   - 画面右側の「Switch コントローラー」エリアが**「検出済み」**になり、スティックやボタンの動きがリアルタイムに反映されることを確認します。
3. toio の電源を入れます。
4. 画面左側の **「toio に接続する」** ボタンをクリックします。
5. ブラウザのポップアップ画面が表示されるので、一覧から **「toio Core Cube」** を選択して「ペアリング」をクリックします。
6. 接続に成功すると、toio から接続音が鳴り、LEDが緑（デフォルト）に点灯します。

## 操作方法

### 🎮 コントローラーでの操作
- **左スティック (または右スティック):**
  - **前進 / 後退 / 旋回:** スティックを倒した方向に toio が走ります。
- **アクションボタン:**
  - **A ボタン:** LEDが緑に光り、接続音が鳴ります。
  - **B ボタン:** LEDが赤に光り、ダメージ音が鳴ります。
  - **X ボタン:** LEDが青に光り、レベルアップ音が鳴ります。
  - **Y ボタン:** LEDが黄に光り、コイン音が鳴ります。

### ⌨️ キーボードでの操作 (コントローラーがない場合)
コントローラーが手元にない場合や認識しない場合でも、キーボードで操作できます。
- **W キー / ↑ キー:** 前進
- **S キー / ↓ キー:** 後退
- **A キー / ← キー:** 左旋回
- **D キー / → キー:** 右旋回
- **1, 2, 3, 4 キー:** それぞれ LEDカラーチェンジ ＋ 効果音が鳴ります。

## トラブルシューティング

- **「toio に接続する」ボタンを押してもデバイスが見つからない**
  - PCの Bluetooth 機能がオンになっているか確認してください。
  - toio の電源が入っているか（ランプが点滅しているか）確認してください。
  - すでに他のアプリ（スマホアプリなど）に toio が接続されている場合は、一度接続を切ってください。

- **コントローラーが認識されない**
  - ブラウザの仕様上、コントローラーのボタンを1回以上押さないと Gamepad API が接続を検知しません。コントローラーのボタンをいくつか押してみてください。
  - Windows 側でコントローラーが正しく接続されているか（コントロールパネルの「ゲームコントローラーの設定」で入力が反映されるか）確認してください。

- **Web Bluetooth がサポートされていないとエラーが出る**
  - Safari、Firefox、Internet Explorer では Web Bluetooth は動作しません。必ず **Chrome** または **Edge** をご使用ください。
  - アドレスバーのURLが `http://localhost:3000` になっているか確認してください。IPアドレス（例: `http://192.168.x.x`）やファイルパス（`file:///...`）で開くと、Web Bluetooth API はセキュリティ制限により無効化されます。
